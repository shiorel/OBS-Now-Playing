# Contributing

Thanks for helping improve OBS Now Playing.

1. Open an issue describing the bug or proposed change.
2. Fork the repository and create a focused branch.
3. Keep the widget compatible with a 400×100 OBS Browser Source.
4. Preserve loopback-only networking and avoid adding telemetry or required accounts.
5. Test Turkish and English UI, idle hiding, and at least one Windows media source.
6. Submit a pull request with a clear summary and verification notes.

AI-assisted contributions are welcome. Please disclose material AI assistance and review generated code before submitting it.
