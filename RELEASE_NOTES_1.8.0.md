# OBS Now Playing 1.8.0

## New themes

Three community-created alternative themes were integrated without replacing the original Dynamic Neon design:

- Dynamic Neon
- Minimal Clean Dark
- Retro Synthwave
- Cyberpunk Neon Glass

The desktop control panel now includes a **Widget theme** selector. Theme selection is stored in `config.json` and reaches an open OBS Browser Source through the existing live configuration refresh.

## Compatibility

- Shared HTML element IDs and API routes are unchanged.
- TR/ENG language synchronization is preserved.
- Long-title marquee, idle hiding, album artwork, artist artwork, service badges, timeline, and optional controls remain available.
- All four themes were validated at 1200×300 (the same 4:1 ratio as the recommended 400×100 OBS source).
