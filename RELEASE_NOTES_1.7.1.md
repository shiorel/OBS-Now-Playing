# OBS Now Playing 1.7.1

First public open-source release.

## Highlights

- Portable, installer-free Windows application
- Modern bilingual TR/ENG control panel
- Live widget-language synchronization with the selected UX language
- Spotify, Apple Music, YouTube Music, Deezer, and browser-media priority
- Deezer artwork lookup with iTunes Search API fallback
- Dynamic Neon 400×100 OBS widget
- Long-title marquee animation
- Fixed `Hide widget when idle` behavior and live setting refresh
- System-tray operation and copyable OBS URL

## Installation

1. Download `OBS-Now-Playing-1.7.1-Windows.zip`.
2. Extract it to a normal writable folder.
3. Run `OBSNowPlaying.exe`.
4. Follow the setup steps shown in the control panel.
